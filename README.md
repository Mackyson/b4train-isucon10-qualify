# 名古屋大学DBグループB4研修｜ISUCON10予選問題

## ディレクトリ構成

- `bin`: 問題解答時に利用すると便利なスクリプト群．
- `bottleneck_analysis`: スクリプトによって自動生成されるボトルネックの解析結果．
- `conf`: 各種設定ファイル．スクリプトによって各サーバへ反映．
- `server_info`: 実施環境の基本的な性能測定結果．
- `webapp`: アプリの実装コード．

## 想定環境

- OS: Ubuntu 20.04 LTS
- DB: PostgreSQL 12
- Webサーバ: Nginx
- アプリ実装言語: Go
